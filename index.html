STOCK PRICE PREDICTION SYSTEM

Developed a stock price prediction platform using Django, integrating linear regression and LSTM models for forecasting. The system allows users to access real-time stock data and news from Yahoo Finance. Users can utilize linear regression for free and access LSTM predictions after a subscription. Key features include user account management, feedback submission, and viewing stock-related information. Admin functionalities include managing user accounts, visualizing website usage, and reviewing user feedback. The project employs MySQL for data storage and emphasizes a user-friendly experience with a focus on accurate stock predictions.

Project 3

RT Office Management System

RT Office Management System

Developed a web-based management system for the Regional Transport Office (RTO), hosted on secure servers. This system enables remote access, simplifies maintenance, and consolidates inventory management from a central database. It supports various RTO functions, including the issuance of driving licenses, learner's licenses, vehicle registrations, ownership transfers, and permits. A significant innovation is the on-the-spot issuance of driving licenses and the implementation of the Computer Aided Learner's Licensing System (CALLS) for instant learner's license issuance. This project streamlines the office's operations, improving record-keeping and increasing employee efficiency.
